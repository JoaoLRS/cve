# CDU-02: Tela de imagem

1. Apos o usuario ter clicado no botão `Imagem` de acordo com o [CDU-01](cdu-01.md) a margem do layout da aplicação ira expandir para que seja posivel a visualização da tela de `Imagem`. Com isso ira conter os seguintes componentes:

    1.1 - Titulo: `Imagem`.

    1.2 - Botão: `Importar` **(o botão `Importar` sempre sedra visivel mesmo já tendo dados no banco de dados)**

2. Se conter dados no banco de dados, deve ser mostrado para o usuario **1 ou n** `cards` contendo as imagens que já foram importadas.

3. Ao clicar em algum `card` mudara para uma outra tela de `procesamento de imagem` referente ao [CDU-03](cdu-03.md).

4. No final do sistema contera um botão `Voltar`, ele sera responsavelo por redirecionar para a tela anterior que nesse casso sera o [CDU-01](cdu-01.md).
